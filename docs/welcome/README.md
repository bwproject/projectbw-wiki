## ⚡ Добро пожаловать в ProjectBW

::: tip О проекте
ProjectBW — это игровая сеть нового поколения, объединяющая несколько игровых миров и сообществ.
:::

Мы создаём единую экосистему, где каждый игрок может выбрать свой формат игры — от классического выживания до кастомных режимов и ивентов.
